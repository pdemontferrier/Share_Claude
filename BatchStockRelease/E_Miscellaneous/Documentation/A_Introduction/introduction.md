# Introduction

## Contexte
L’application **BatchStockRelease** s’inscrit dans la démarche de digitalisation de l’atelier de fabrication de Le Vérandier by Tryba.
Elle gère le processus de **sortie de stock et d’approvisionnement des postes de découpe**, en s’intégrant dans l’écosystème ERP.

## Objectifs
- Optimiser les mouvements de stock.
- Gérer les approvisionnements des postes de découpe.
- Garantir la traçabilité complète des opérations.

## Technologies
- **.NET 8 / WPF**
- **Entity Framework Core / Pomelo MySQL**
- **Clean Architecture, DDD, MVVM, CQRS**
